# 🏛️ Institutional COT Dashboard
تطبيق احترافي لتحليل تقارير التزام المتداولين (COT) من هيئة CFTC باستخدام Python و Streamlit.

## المميزات
- جلب تلقائي للبيانات من موقع CFTC الرسمي.
- تحليل العملات الرئيسية، الذهب، والمؤشرات.
- حساب "التقييم المؤسسي" بناءً على صافي المراكز.

## طريقة التشغيل
1. `pip install -r requirements.txt`
2. `streamlit run app.py`
